Documentation

Getting Started

At first, you need to include this library and its style sheet in your html file.

<script defer type="text/javascript" src="game-planner.js"></script>
<link rel="stylesheet" type="text/css" href="game-planner.css"></link>
<link rel="stylesheet" type="text/css" href="https://cdn.jsdelivr.net/npm/bootstrap@4.3.1/dist/css/bootstrap.min.css"></link>

JS Object Explaination - Event Object (Internal Object)

Attribute Type Description Default Value Required Note
player1 String one player of the game N/A true N/A
player2 String Another player of the game N/A true N/A
startTime Date the event's start time N/A true When creating a Date object in js, month value starts from 0. AKA month for January is 0
eventID Integer Unique id for each event N/A true eventID will be generated automatically during creation, and will be used when removing events
duration Integer Duration of the event 3 true N/A
type String Event type N/A false Eg. basketball, tennis
location String Event Location N/A false Eg. Toronto
gamename String Event Game Name N/A false Eg. NBA
color String the color of this event block to be displayed #E0FFFF true must be in format '#xxxxxx'
detail String Event Details, other information you want to include in the event to be displayed N/A false N/A
win String The player name that wins the game N/A false N/A
resultdetail String More result details you want to include in the event to be displayed N/A false N/A

JS Object Explaination - GamePlanner Object

Attribute Type Description Default Value Required Note
events List[Event] A list of events that are will be displayed [] true N/A
numberOfEvents Integer Number of events that have been added. 0 true For generating event ID. The number will not decrease when removing events.
tableName String The game planner's title Game Planner true N/A
latestTime Date The latest startTime of all events N/A false No need to input, library will calculate itself. 'null' when no event
earliestTime Date The earliest startTime of all events N/A false No need to input, library will calculate itself. 'null' when no event

JS Object Explaination - GameRenderer Object

Attribute Type Description Default Value Required Note
filterVariables List[String] a list filter variables included for users to filter events. ['NA', 'player'] true N/A
filteredEventList [Event] A list of filtered events GamePlanner.events true Only support following values in list: 'NA', 'player', 'type', 'location', 'gamename', 'win', 'result-detail', 'detail', 'duration'
container HTML Node The location where the table is added in a HTML file N/A true N/A
gamePlanner GamePlanner GamePlanner object that includes all event data that will be displayed N/A true N/A
showByWeeks Boolean if default display is the week table N/A true N/A
startDate Date The startDate to display in the table. Will be the day to display for Day table, and the Sunday of displayed week for Week table Today for Day table, and Sunday for this week for Week table true Will be updated when input Date or click 'last' and 'next'

Constructor

new GamePlanner()

This function will create a Default GamePlanner.

const gamePlanner = new GamePlanner();

new GameRenderer(gamePlanner, container)

This function will create a Default GameRenderer, and the GameRenderer will display the table as a child of the container.

let container = document.createElement('div');
let gameRenderer = new GameRenderer(gamePlanner, container);

Common Methods

GamePlanner.setTableName(tableName)

This function will set displayed table name.
tableName: String, tableName is the title you want to set for the event planner table. The table name set will be shown at the top of the table.

gamePlanner.setTableName('Example Game Event Planner');

GamePlanner.getEventList()

This function will get all events in the library

gamePlanner.getEventList();

GamePlanner.addEvent(player1, player2, startTime, options)

This function will add an event to the library

player1: String, one player of the game
player2: String, another player of the game
startTime: Date, the event's start time
options: Dict, a dictionary that involves optional parameters (All following options.xxx are optional)
options.duration: Integer, duration of the event, default = 3
options.type: String, the type of the event. Eg. tennis
options.location: String, the location of the event
options.gamename: String, game name of the event. Eg. NBA
options.detail: String, some additional information about this event. Eg. who holds this event, what is the more attracting point.
options.color: String, should be format '#xxxxxx', which decides the color of this event block to be displayed. Default = '#E0FFFF'
options.result: Dict, a dictionary that includes event result information, includes who wins and more details
options.result.win: String, the player that wins the event
options.result.detail: String, more details about the event result. Eg. detailed scores

gamePlanner.addEvent('player1', 'player2', new Date(2022, 3, 16, 5, 0, 0, 0), {'type': 'tennis', 'location': 'toronto', 'gamename': 'atp', 'duration': 3, 'result': {'win': 'player1', 'detail': 'player1 wins 1,3,5 round'}, 'detail': 'Player1 and player2 are both really excellent tennis players, this match is a strong-strong fight'});

GamePlanner.removeEvent(eventID)

This function will remove event from library based on eventID (eventID will be auto-generated during creation)
eventID: Integer, eventID of the event that we want to remove.

gamePlanner.removeEvent(gamePlanner.getEventList()[0].eventID);

GameRenderer.addFilterVariables(filter_list)

This function will add filter variables that are for users to filter events.

filter_list: List[String], a list filter variables included for users to filter events. Default filter_list=['NA', 'player']. Only support following values in list: 'NA', 'player', 'type', 'location', 'gamename', 'win', 'result-detail', 'detail', 'duration'

gameRenderer.addFilterVariables(['type', 'location', 'gamename', 'win', 'result-detail', 'detail', 'duration']);

GameRenderer.removeFilterVariables(filter_list)

This function will remove filter variables that are for users to filter events.
filter_list: List[String], filter variables to be removed for users to filter events. Only support following values in list: 'NA', 'player', 'type', 'location', 'gamename', 'win', 'result-detail', 'detail', 'duration'

gameRenderer.removeFilterVariables(['type', 'location']);

GameRenderer.setTableType(showByWeeks)

This function decides if diplay Week Table by default.
showByWeeks: Boolean, if we want to show the week table first. Default=true.

gameRenderer.setTableType(true);

GameRender.display()

This function can display the final table based on previous setups..

gameRenderer.display();
