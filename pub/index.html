Sport Games Planner

This library aims at displaying sports games events in a decent way.


A Week table and a Day table are included.

It is obvious that the Week table diplay events by weeks, while the Day table display events by Days and with more details.


The events can be filtered and change displayed color.


You could navigate to other pages for more details.
